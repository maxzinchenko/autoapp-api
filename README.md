# AutoApp Api

Using the [Nest](https://github.com/nestjs/nest) framework with [Sequelize](https://github.com/sequelize/sequelize) ORM.

## Packages installation

```bash
$ yarn
```

## Running the app

```bash
# development mode
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Running the tests

```bash
# unit tests
$ yarn test

# test coverage
$ yarn test:cov
```

## Checking for the code style exceptions

```bash
# eslint check
$ yarn lint
```
